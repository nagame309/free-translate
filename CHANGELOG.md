# 更新日誌

所有重要的專案更新都會記錄在此文件中。

格式基於 [Keep a Changelog](https://keepachangelog.com/zh-TW/1.0.0/)，
並且本專案遵循 [語義化版本](https://semver.org/lang/zh-TW/)。

---

## [2.0.0] - 2026-01-28

### ✨ 重大更新

- **專案更名** - 正式更名為「浮譯 FreeTrans」。
- **AI 翻譯支援** - 整合 OpenAI 及相容 API (如 OpenRouter, Ollama, DeepSeek)。
- **主題切換** - 支援深淺模式 (Light/Dark Mode) 手動切換。
- **UI 布局重構** - 重新設計 Popup 與 Options 頁面，更加簡潔優雅。

### 🚀 新增功能

- **語言自動偵測標籤** - 翻譯結果現在會顯示偵測到的來源語言名稱。
- **測試連線功能** - 在設定頁面可即時測試 AI 服務連線狀況。
- **自動儲存** - 翻譯開關切換後立即自動儲存。
- **進階 AI 設定** - 支援自訂 System Prompt、User Prompt 與 Temperature。

### 🔧 修復與優化

- **路徑拼接修正** - 修復 AI API URL 重複疊加 `/chat/completions` 的問題。
- **CORS 最佳化** - 優化對本地 AI 服務 (如 Ollama) 的連線相容性。
- **Namespace 統一** - 將程式碼內部命名空間統一為 `FreeTrans`。
- **Bug Fix** - 修復 Popup 開啟時會自動跳出「設定已儲存」提示的問題。

## [1.0.0] - 2026-01-28

### 🎉 初始版本

這是浮譯的第一個正式版本!

### ✨ 新增功能

- **即時翻譯** - 選取文字自動顯示翻譯結果
- **智能過濾** - 自動跳過純中文內容(80%以上中文字元)
- **一鍵複製** - 點擊翻譯內容即可複製到剪貼簿
- **多語言支援** - 支援英、日、韓、法、德、西等多種語言
- **自動翻譯開關** - 可在設定面板開啟/關閉功能
- **語言選擇** - 可自訂來源語言和目標語言

### 🎨 設計特色

- **毛玻璃浮動視窗** - 採用 Glassmorphism 設計風格
- **Catppuccin 配色** - 使用 Latte 淺色主題
- **流暢動畫** - 淡入淡出與縮放效果
- **智能定位** - 浮動視窗自動避開螢幕邊緣
- **動態下拉箭頭** - 選單展開時箭頭向上

### ⚡️ 效能優化

- **防抖機制** - 150ms 延遲避免頻繁請求
- **最少字元限制** - 過濾過短的選取內容
- **滾動偵測** - 自動隱藏浮動視窗
- **錯誤處理** - 友善的錯誤提示訊息

### 🛠 技術細節

- 使用 Chrome Extension Manifest V3
- 純原生 JavaScript,無框架依賴
- 整合 Google Translate API
- 使用 MapleMono NF CN 等寬字體
- 響應式設計,適應不同螢幕尺寸
