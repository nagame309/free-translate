# 浮譯 (FreeTrans) v2.0

感謝下載浮譯!

## 📦 安裝步驟

### 方法一:手動安裝 (推薦)

1. 下載並解壓縮 `free-trans-v2.0.zip`
2. 開啟 Chrome 瀏覽器
3. 前往 `chrome://extensions/`
4. 開啟右上角的「開發人員模式」
5. 點擊「載入未封裝項目」
6. 選擇解壓縮後的資料夾
7. 完成!享受優雅的翻譯體驗 ✨

### 方法二:拖曳安裝

1. 下載 `free-translate-v1.0.zip` 並解壓縮
2. 前往 `chrome://extensions/`
3. 開啟「開發人員模式」
4. 將資料夾拖曳到擴充功能頁面
5. 完成!

## 🎯 快速開始

1. **選取文字** - 在任何網頁選取文字(至少 3 個字元)
2. **自動翻譯** - 浮動視窗自動顯示翻譯結果
3. **點擊複製** - 點擊翻譯內容即可複製

## ⚙️ 設定

點擊工具列的擴充功能圖示可以:
- 調整來源語言和目標語言
- 開啟/關閉自動翻譯功能

## 📝 注意事項

- 選取的文字需至少 3 個字元
- 純中文內容(80%以上)會被自動跳過
- 在 `chrome://` 開頭的頁面無法使用
