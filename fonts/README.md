# Maple Mono NF CN 字體安裝說明

## 字體來源

Maple Mono NF CN 是一個支援中文的等寬字體，包含 Nerd Font 圖標。

## 安裝方式（擇一）

### 方式一：使用系統已安裝的字體

如果您的系統已經安裝了 Maple Mono NF CN 字體，擴充功能會自動使用。

檢查方式：
1. Windows: 開啟「設定」→「個人化」→「字型」
2. 搜尋 "Maple Mono" 或 "MapleMono"

### 方式二：下載字體文件到擴充功能

1. 下載 Maple Mono NF CN 字體：
   - 官方網站：https://github.com/subframe7536/maple-font
   - 或使用 Nerd Fonts：https://www.nerdfonts.com/

2. 將字體文件放到 `fonts/` 目錄：
   - `MapleMono-NF-CN-Regular.ttf`
   - `MapleMono-NF-CN-Bold.ttf` (可選)

3. 重新載入擴充功能

### 方式三：使用替代字體

如果無法使用 Maple Mono NF CN，擴充功能會自動降級到以下字體：
- Cascadia Code
- Fira Code
- Consolas
- 系統默認等寬字體

## 字體效果

使用 Maple Mono NF CN 字體後：
- ✅ 更好的中文顯示效果
- ✅ 等寬字體，對齊美觀
- ✅ 支援 Nerd Font 圖標
- ✅ 優秀的可讀性

## 當前狀態

目前擴充功能已配置為優先使用 Maple Mono NF CN 字體。如果系統沒有此字體，會自動使用替代方案，不影響功能使用。
